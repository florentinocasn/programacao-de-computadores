# 💻 Programação de Computadores

Resolução de exercícios práticos da disciplina Programação de Computadores, ministrada pela Me. Joanacelle Caldas de Melo. 
A disciplina tem como objetivo a formação de profissionais capacitados a solucionar problemas complexos 
por meio da lógica computacional utilizando a linguagem de programação Python.
